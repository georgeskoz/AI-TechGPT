# Customer Portal - TechersGPT

Customer-facing application for accessing technical support services.

## Features

- AI-powered chat support
- Phone support booking
- Service provider marketplace
- Service tracking
- Profile management

## Running the App

```bash
# From project root
npm run dev:customer
```

Access at: http://localhost:5002

## Key Pages

- `/chat` - AI chat support (default home page)
- `/dashboard` - Customer dashboard
- `/services` - Browse support services
- `/marketplace` - Find service providers
- `/profile` - Manage profile

## Registration

New customers can register at `/register`

## Building

```bash
npm run build:customer
```

Output: `dist/customer/`
