# Service Provider Portal - TechersGPT

Portal for technicians and service providers to manage their business.

## Features

- Job management (active, pending, completed)
- Earnings tracking
- Customer communication
- Schedule management
- Profile and certifications

## Running the App

```bash
# From project root
npm run dev:provider
```

Access at: http://localhost:5003

## Key Pages

- `/provider/dashboard` - Provider dashboard
- `/provider/jobs` - Job management
- `/provider/earnings` - Earnings and payouts
- `/provider/profile` - Profile management
- `/provider/messages` - Customer messages

## Registration

Service providers register at `/provider/register`

## Building

```bash
npm run build:provider
```

Output: `dist/provider/`
