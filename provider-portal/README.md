# TechersGPT Service Provider Portal

## Overview
This is a dedicated portal for service providers/technicians, separate from the customer and admin interfaces.

## Features
- Dedicated provider authentication
- Job management dashboard
- Earnings tracking
- Real-time notifications
- Profile management
- Direct communication with customers

## Tech Stack
- React + TypeScript
- Vite
- Wouter for routing
- TanStack Query
- Supabase (PostgreSQL)
- Shadcn UI + Tailwind CSS

## Deployment
This portal can be deployed as a standalone application:
1. Copy this directory to a new Replit project
2. Update the DATABASE_URL to point to the same Supabase instance
3. Deploy at providers.techersgpt.com
